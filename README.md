# 🌍 TripNest

TripNest is a modern travel planning web application built with React and Vite. It helps users discover destinations, explore nearby places, get travel recommendations, and plan routes efficiently through an intuitive and responsive interface.

## 🚀 Live Demo

🔗 https://tripnest-react.vercel.app/

## ✨ Features

- 🏠 Interactive Home Page
- 🔐 Login & Registration Interface
- 📍 Nearby Places Discovery
- ⭐ Travel Recommendations
- 🗺️ Route Mapping & Navigation
- 📝 Travel Planning Form
- 📱 Responsive Design
- ⚡ Fast Performance with Vite

## 🛠️ Tech Stack

### Frontend
- React.js
- Vite
- JavaScript (ES6+)
- HTML5
- CSS3

### Deployment
- Vercel

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/Tekikathik/TripNest.git
```

Navigate to the project directory:

```bash
cd TripNest
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```

Preview production build:

```bash
npm run preview
```
